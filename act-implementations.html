Accessibility Conformance Testing (ACT) Rules Implementations

This page tracks implementations of the Accessibility Conformance Testing (ACT) Rules Format 1.0 for the Candidate Recommendation (CR) phase.

Instructions

Document results in the implementation column with a link to the rule to be used to satisfy the particular requirement and whether it passes, fails, or is in progress. Here's a few examples:

  • Links Have An Accessible Name pass
  • Images Have An Accessible Name fail
  • Valid Body Lang Attribute in progress

ACT Rules Format Implementations

The following tables document the implementations of the various normative requirements in the ACT Rules Format 1.0. Rules must meet the exit criteria for the standard to progress to CR status. These exit criteria are also included in the Exit Criteria section of the PR version of the spec.
